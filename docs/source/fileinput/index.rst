File Input Modules
===================
Contains functions and codes related to:

* Reading a MRM transition name data file
* Validation of the file and the content in the file
* Extracting raw values in the file and convert it into a pandas dataframe
* Reading the MS Template Creator Excel Macro sheet
* Validation of the sheets and the contents in the sheet
* Extracting/Combining values in sheets to create

  * Transition Name Annotation dataframe 
  * ISTD Annotation dataframe
  * Sample Annotation dataframe
